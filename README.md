## :page_with_curl: General Information
MemStego is generaly about images, text and memes! By using steganography methods, in particular the LSB(Less segnificant byte) the image may be manupulated in a way to keep a secret message.

## :gear: Setup
- Clone the project repository
- Run <kbd>pip install -r requirements.txt</kbd> (Python 2), or <kbd>pip3 install -r requirements.txt</kbd> (Python 3)
<p align="center">
  <img src="https://i.imgur.com/RWwb4aN.png" width="100%" alt="---------------------------------------">
</p>

## :wrench: Usage

 #### **Executable file:** Go to the directory, where executable is placed, and run <kbd>./memstego</kbd>.
 #### **All project files:** Go to the project directory and run<kbd>python3 ./memstego.py</kbd>

## :old_key: Features
- Embed encrypred message in image
- Fetch and decode a hidden messages from a image
- Generate random meme and embed the secret message
